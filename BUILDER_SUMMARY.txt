📋 USAGE INSTRUCTIONS
═══════════════════════════════════════════════════════════════════════════════════

   OPTION 1: Clone & Run Locally
   ─────────────────────────────
   $ git clone https://github.com/kapilexofficial/pool-services-hub.git
   $ cd pool-services-hub
   $ npm install
   $ cp .env.example .env.local
   [Add Supabase credentials]
   $ npm run dev

   OPTION 2: Import to Lovable
   ─────────────────────────────
   1. Go to lovable.dev
   2. Click "Create Project"
   3. Choose "Import from GitHub"
   4. Enter: kapilexofficial/pool-services-hub
   5. Follow SETUP.md for database configuration

   OPTION 3: Deploy to Vercel
   ─────────────────────────────
   1. Push to GitHub (already done ✓)
   2. Connect repository to vercel.com
   3. Set environment variables in Vercel dashboard
   4. Deploy!
